# 2026.7-9-v5 审稿意见驱动修改说明

## 一句话论证

在迭代式无监督说话人验证中，CGPL 使用两个异构语音预训练模型的对齐聚类一致性筛选初始伪标签；现有 VoxCeleb 结果支持这种一致性作为实用的赋值稳定性信号，但结论限于当前标签辅助离线配置、无标签伪标签构建与梯度优化协议。

## 本轮修改原则

- 未新增、补造或外推任何实验、指标、随机种子结果或计算成本。
- 能由现有正文和表格支持的问题直接修订。
- 无法通过写作消除的问题在正文中透明披露并收缩主张。
- 缺失但无法从项目文件确认的实现参数不作猜测，列在“仍需作者输入”中。

## 已完成的主要修改

1. **标题与摘要**
   - 标题删除缺少跨域证据支撑的 `Robust`。
   - 摘要改为“问题—方法—关键伪标签证据—EER—标签使用边界—适用范围”的论证链。
   - 将“三次 refinement iterations”统一为“三次 training iterations”。
   - 删除“严格 SOTA”“更少迭代”“与监督系统相当”等证据不足的概括。

2. **引言与贡献定位**
   - 将 cross-model agreement 从“正确性/可靠性证明”限定为“assignment-stability signal”。
   - 将核心贡献聚焦于异构分区共识筛选及按共识状态分配训练目标。
   - 删除正文并未提供的 complexity analysis 和完整 sensitivity analysis 预告。

3. **方法可读性**
   - 明确 Iteration 1 是 consensus-guided initialization 加首次模型训练；后续轮次才称 refinement iterations。
   - Algorithm 2 的后续伪标签生成明确关联到 segment-level modal-prediction 和 majority-acceptance 规则。
   - 明确 HNM 只用于第一轮的 cross-view-disagreement subset，并说明 same-speaker false negative 无法被小 batch 完全消除。

4. **实验协议透明度**
   - 新增 `Evaluation Protocol and Claim Scope`。
   - 明确 Vox1-O 用于 PTM 层和对比损失权重分析，因此 Vox1-O 是 development-informed，而非 untouched test set。
   - 明确 VoxCeleb2 身份标签用于离线聚类配置比较和伪标签指标报告，但不进入 CGS、PLM、DLNAS、HNM 或参数更新。
   - 明确当前结果是 point estimates，缺少 repeated-run uncertainty 和 minDCF。
   - 明确与公开方法的比较为 descriptive comparison，而非统一资源和训练协议下的受控排名。

5. **指标与数值一致性**
   - 保留作者修正后的 UniSpeech-SAT/Infomap F1=`0.478`，并同步后续表格与正文。
   - 增加 F1 的调和平均公式。
   - 将 Purity 明确为 macro-averaged cluster Purity，并说明其受簇数和簇规模分布影响。
   - 明确 full partitions 与 69.19% selected subset 的 coverage 不同，因此不能把指标差异完全归因于 agreement。

6. **结果与消融解释**
   - 将 “outperform/demonstrate” 等强表述改为具体表内观察和有边界的解释。
   - 对 PLM、DLNAS 和普通 CL 的小幅 EER 变化增加单次运行限制。
   - 明确 DLNAS 消融无法区分 residual-noise mitigation 与 confidence-dependent reweighting。
   - 将 Vox1-O/E/H 的一致表现限定为同一 benchmark family 内的观察，不外推到跨语种或跨域鲁棒性。

7. **Discussion and Limitations**
   - 新增独立讨论章节，覆盖 coverage confounding、机制替代解释、测试集调参、比较公平性、统计不确定性、Hungarian 一对一粒度限制、PLM 极值阈值敏感性、HNM false negatives 和同域评估边界。

8. **Conclusion 与投稿格式**
   - 结论压缩为贡献、决定性证据和适用边界，不再重复逐项结果。
   - 清理示例页眉和重复 `cite` 包。
   - 参考文献样式改为 `IEEEtran`；由于正文目前使用手写编号而不是 `\cite{}`，暂时保留 `\nocite{*}` 以生成文献表。

## 仍需作者输入或实验记录核实

以下问题不能在“不新增实验且不猜测实现”的条件下通过写作解决：

1. DLNAS 的 EMA momentum $v$ 及初始值 $\delta_0$。
2. AdamW weight decay 和其他未报告的优化器参数。
3. WavLM、UniSpeech-SAT 的确切 checkpoint、模型规模、层编号约定和 utterance pooling 实现。
4. Infomap 图构建的近邻数、边阈值、图剪枝和随机性设置。
5. PLM 质心使用的具体 embedding 来源，以及多对簇同时满足阈值时的合并顺序、连通分量规则或质心重计算规则。
6. Precision/Recall 的具体聚类定义（例如 pairwise 或其他实现）；当前仅能确定 F1 为两者的调和平均。
7. 训练轮次之间模型是否延续参数、分类头如何适配新伪标签集合，以及两类 mini-batch 的具体调度方式。
8. 多随机种子均值/标准差、trial-level 置信区间、minDCF 和计算成本；这些需要已有日志或新增实验。
9. PTM 预训练数据与 VoxCeleb 的潜在重叠说明。
